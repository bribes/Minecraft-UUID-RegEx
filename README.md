# Minecraft UUID RegEx

### RegEx for both dashless and dashed UUID provided by drainer / lucky swede

#### ```/^[A-F\d]{8}-[A-F\d]{4}-4[A-F\d]{3}-[89AB][A-F\d]{3}-[A-F\d]{12}$/i```

### RegEx for UUID with dashes

#### ```/^[A-F\d]{8}-[A-F\d]{4}-4[A-F\d]{3}-[89AB][A-F\d]{3}-[A-F\d]{12}$/i```

### RegEx UUID without dashes

#### ```/^[A-F\d]{8}[A-F\d]{4}4[A-F\d]{3}[89AB][A-F\d]{3}[A-F\d]{12}$/i```
